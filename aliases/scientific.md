# Scientific Computing Aliases

This page collects **widely-used** import aliases across the scientific Python stack. Conventions like `np`, `pd`, `plt`, and `sns` are broadly recognized in tutorials and docs. [[1](https://pytutorial.com/importing-python-libraries-pandas-requests-os/)] [[2](https://apxml.com/courses/data-visualization-matplotlib-seaborn/chapter-1-intro-data-visualization-python/importing-plotting-libraries)] [[3](https://seaborn.pydata.org/tutorial/introduction)]

## Matplotlib (pyplot) → `plt`

**Import**

```python
import matplotlib.pyplot as plt
```

**Notes**

Using plt for matplotlib.pyplot is a well-established convention in data viz materials.

---

## NumPy → `np`

**Import**

```python
import numpy as np
```

**Notes**

The np alias is a near-universal convention across learning resources and examples.

---

## pandas → `pd`

**Import**

```python
import pandas as pd
```

**Notes**

pd is the standard alias in tutorials and documentation.

---

## Seaborn → `sns`

**Import**

```python
import seaborn as snsShow more lines
```

**Notes**

The seaborn docs themselves show import seaborn as sns and call it the conventional shorthand.

---
